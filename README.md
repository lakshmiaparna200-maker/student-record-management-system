Expense Tracker

A simple Python Expense Tracker project built using File Handling and Functions. This project helps users record daily expenses, view saved expenses, and calculate the total amount spent.

Features
Add new expenses
View all saved expenses
Calculate total expenses
Menu-driven interface
Stores data in a text file

How to Run
Open the project folder in VS Code.
Open the terminal.
